ubuntu安装遇见的问题

Posted at 2021-01-28

  • ubuntu安装完成后，远程ssh连接报错
port 22: Connection refused

用本机判断是否安装ssh服务

ssh localhost

会报ssh: connect to host localhost port 22: Connection refused

使用命令查看

ps -e | grep ssh

只有一个进程ssh-agent或者一个进程都没有

安装openssh-server

sudo apt-get install openssh-server

安装完成后再次查看

ps -e | grep ssh

会出现sshd的进程，此时再用ssh进行连接就可以连接上了

  • Ubuntu配置更新源

vim /etc/apt/sources.list

deb http://mirrors.163.com/ubuntu/ oneiric main restricted universe multiverse
