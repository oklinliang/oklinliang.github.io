习惯mysql后初次用orcale的坑

Posted at 2021-01-21

查询时间

orcale:a = to_date( '2021:01:15 00:00:00', 'yyyy-mm-dd hh24:mi:ss' )
mysql:a = ('2020-01-15')

分页查询第3～第5条数据

orcale:SELECT ro FROM (SELECT ROWNUM ro FROM 表名) WHERE ro >=3 AND ro <= 5
mysql:select * from 表名 limit 2,3

查询数据库时间

orcale:SELECT SYSDATE FROM dual
mysql:select now()

定义变量进行查询

orcale:
declare @id varchar(20);
set @id='123'
select * from 表名 where id = @id;
mysql:
set @a=123;
select * from 表名 where id = @a;
