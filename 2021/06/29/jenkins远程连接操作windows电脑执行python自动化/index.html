jenkins远程连接操作windows电脑执行python自动化

Posted at 2021-06-29

准备jenkins
远程连接主机安装jdk
连接jenkins

点击jenkins-系统管理-节点管理-新建节点

输入节点名称，选择固定节点确定
在页面创建对应工作目录及条件

点击创建按钮

打开该节点，点击agent.jar下载jar包，放到对应工作目录

新建文本，更改名字为Launch.cmd
内容输入页面上的内容，启动cmd脚本

java -jar agent.jar -jnlpUrl http://192.168.1.94:8888/computer/my_test/jenkins-agent.jnlp -secret 3d3e78d2bff2d3da2645db38da3e5f6457396eecd59465dbe75a93d7d7f49a00


此时jenkins就会显示连接成功的页面
